# Wulfpack Queue Presets v0.0.9

Diese Presets sind für die Direkteingabe `Q1`, `Q2`, `Q3`, `Q4`, `Q5`, `Q6` im Queue-Generator gedacht.

## Globale Regeln

- Wiederholbarer Wrapper in jedem Queue-Block.
- `Collect Quest Rewards` nur in der letzten Queue.
- `Buff > Shield`: `2hr`, `shield_on_everyday`, danach `Gathering Search Menu`.
- `Buff > Gather Speedup`: danach `Gathering Search Menu` + `Alliance Gathering`.
- `Collect Base Resources`: Serum, Food, Wood, Steel, Gas + Antiserum-Limit-Meldung bestätigen.
- Zombies max Level 40; `39-59` ist ungültig.

## Wiederholbarer Wrapper in jeder Queue

- `speedup_help`
- `nova_daily_praise`
- `nova_research`
- `noah_tavern`
- `radio_quiz`
- `collect_base_resources`
- `alliance_research`
- `alliance_gifts`
