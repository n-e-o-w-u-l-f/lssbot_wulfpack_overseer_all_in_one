# IRC Format

Rohlog:

```text
[Farms] 20:00:13 [INFO] Executing a total of 1 scripts
```

Deutsch interpretierte Events:

```text
[Farms] aktive Script-Queue: Commend Leaderboard
[Farms] letzter Charakter erreicht; Task ist fertig
[Farms] Fehler beim Charakterwechsel
[Farms] Emulator-Screenshot fehlgeschlagen; Recovery nötig
```

Der Prefix kommt aus:

```json
"prefix_format": "[{instance}]"
```
